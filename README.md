# PixLP

PixLP is an AI pixel LP diary website. Upload an OOTD photo and a place photo, then create a pixel-style daily album and collect it on a monthly LP shelf.

## Deploy to Vercel

This folder is ready to deploy as a static Vercel project.

1. Push this project to GitHub.
2. Import the GitHub repository in Vercel.
3. Use these settings:
   - Framework Preset: Other
   - Build Command: empty
   - Output Directory: empty
4. Set the Vercel project name to `pixlp`.

The public website link will look like:

```text
https://pixlp.vercel.app
```

## AI Server

The AI server files are kept in this folder for reference, but excluded from the Vercel frontend deployment.
